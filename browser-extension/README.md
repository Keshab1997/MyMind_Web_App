# My Mind Browser Extension

ডেস্কটপ ব্রাউজার থেকে এক ক্লিকে লিংক সেভ করুন।

## ফিচার

✓ সিকিউর লগইন সিস্টেম
✓ অটোমেটিক URL এবং Title ক্যাপচার
✓ থাম্বনেইল ইমেজ অটো ফেচ
✓ কাস্টম নোট যোগ করা
✓ ইউজার-স্পেসিফিক ডাটা সেভ
✓ দ্রুত এবং সহজ

## ইনস্টলেশন (Chrome/Edge/Brave)

1. Chrome ব্রাউজার ওপেন করুন
2. অ্যাড্রেস বারে টাইপ করুন: `chrome://extensions/`
3. ডানদিকের উপরে "Developer mode" অন করুন
4. "Load unpacked" বাটনে ক্লিক করুন
5. এই `browser-extension` ফোল্ডারটি সিলেক্ট করুন
6. এক্সটেনশন ইনস্টল হয়ে যাবে!

## ব্যবহার

### প্রথমবার:
1. এক্সটেনশন আইকনে ক্লিক করুন
2. আপনার My Mind অ্যাপের ইমেইল এবং পাসওয়ার্ড দিয়ে লগইন করুন
3. লগইন সেশন সেভ থাকবে

### লিংক সেভ করা:
1. যেকোনো ওয়েবসাইটে যান
2. এক্সটেনশন আইকনে ক্লিক করুন
3. URL এবং Title অটোমেটিক আসবে
4. চাইলে Note যোগ করুন
5. "Save to My Mind" বাটনে ক্লিক করুন
6. ব্যাস! আপনার My Mind অ্যাপে সেভ হয়ে গেছে

### লগআউট:
- এক্সটেনশন খুলে উপরে ডানদিকে "Logout" বাটনে ক্লিক করুন

## নোট

- আপনার ইমেইল অবশ্যই ভেরিফাইড হতে হবে
- লগইন সেশন ব্রাউজারে সেভ থাকে
- শুধুমাত্র আপনার অ্যাকাউন্টে ডাটা সেভ হবে (RLS সিকিউরিটি)
